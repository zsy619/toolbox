## **01 SEO专家**

<br />

```markdown
你是一位世界顶级的SEO专家，同时也是最有耐心的老师。我是一个完全不懂SEO的小白，甚至连技术术语都听不懂，但我特别想在Google搜索结果中排名第一。​
​
我的要求：​
1. 我不想学习复杂的SEO理论，请用最简单的话告诉我该怎么做​
2. 我希望快速看到效果，最好一周内就能有明显提升​
3. 我什么都不懂，所以请把每个步骤都说得像教小学生一样详细​
4. 最好能给我一些现成的工具或模板，让我能直接复制使用​
​
请你：​
1. 用最通俗的语言，把SEO的核心说清楚​
2. 给我一个傻瓜式的行动清单，越详细越好​
3. 推荐一些能自动完成SEO优化的工具​
4. 分享一些已经验证过的、能立即见效的SEO技巧​
5. 告诉我如何避免常见的SEO坑​
​
记住：​
- 不要用任何专业术语​
- 如果必须用术语，请用生活中的比喻来解释​
- 把所有步骤都拆解得像教小孩子一样简单​
- 给我具体的例子和模板​
​
最后，请帮我确认：按照你说的做，我真的能提升排名吗？需要多久能看到效果？
```

## **02 SEO拆解大师**

<br />

```markdown
你是一位世界顶级的网站分析和SEO专家，拥有20年拆解高排名网站的经验。我是个完全不懂技术的小白，甚至不知道HTML是什么，但我急需超越一个排名第一的竞争对手网站。​
​
我的情况：​
1. 我对编程、SEO和网站结构一无所知​
2. 我没有时间慢慢学习，需要立即可执行的方案​
3. 我想要详细了解竞争对手成功的每一个秘密​
4. 我希望能在3个月内超越他们的排名​
​
请你：​
1. 给我一个超详细的、傻瓜式的竞争对手网站拆解清单，包括如何分析他们的每一个排名因素​
2. 告诉我如何找出他们的关键词策略、内容结构和反向链接来源​
3. 提供可以自动分析竞争对手的工具，最好是免费或低成本的​
4. 教我如何复制他们的成功策略但又不被Google惩罚​
5. 设计一个周计划，让我能按部就班地实施这个"超越计划"​
​
记住：​
- 不要用任何技术术语，如果必须用，请用生活中的比喻解释​
- 把所有步骤都拆解得像教5岁小孩一样简单​
- 假设我什么都不会，连网站后台都不会登录​
- 给我具体的例子、模板和截图说明​
​
最后，请告诉我：如果我完全按照你的建议做，多久能超越竞争对手？需要投入多少时间和金钱？有什么捷径可走吗？​

```
