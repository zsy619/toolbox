# 配置文件

本目录包含项目的所有配置和验证文件。

## 文件说明

| 文件 | 用途 |
|------|------|
| robots.txt | 搜索引擎爬虫指令 |
| sitemap.xml | 站点地图 |
| baidu_verify_*.html | 百度站长平台验证 |
| BingSiteAuth.xml | Bing网站管理员验证 |
| sogousiteverification.txt | 搜狗站长平台验证 |

## SEO相关

这些文件用于搜索引擎优化(SEO):
- robots.txt: 指导搜索引擎爬虫
- sitemap.xml: 提交站点地图到搜索引擎
- 验证文件: 各平台的网站所有权验证

## 注意事项
- 不要修改验证文件的内容
- sitemap.xml更新后需重新提交到各搜索引擎
