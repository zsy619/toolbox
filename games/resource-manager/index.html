🏗️ 游戏说明

🎯 游戏目标

建设和管理一座繁荣的城市，合理分配资源，发展经济，研究科技，让城市不断发展壮大。

🏗️ 建筑类型

• 住房：增加人口容量，提供居住空间
• 农场：生产食物，维持人口需求
• 矿场：开采石材，用于建设
• 伐木场：收集木材，建筑材料
• 市场：增加金币收入
• 工厂：消耗资源生产能源

🎮 操作方式

1. 选择要建造的建筑类型
2. 点击城市网格中的空地建造
3. 管理资源平衡，确保可持续发展
4. 研究科技提升生产效率

💡 策略提示

• 平衡发展各类建筑
• 优先建造资源生产建筑
• 及时研究科技提升效率
• 关注人口和资源的平衡
• 合理规划城市布局

🔬 科技系统

• 研究科技需要消耗金币
• 不同科技提供不同加成
• 科技效果永久生效
• 合理规划研究顺序
