# 🎮 经典单页游戏合集

一个精心制作的经典游戏集合，使用原生Web技术实现，无需安装即可在浏览器中畅玩。

## 🚀 快速开始

```bash
# 进入游戏目录
cd /Volumes/E/JYW/工具箱/games

# 启动本地服务器
python3 -m http.server 8000

# 访问游戏合集首页
open http://localhost:8000
```

## 🎯 游戏清单

### 1. 🐍 贪吃蛇 (Snake Game)
- **路径**: `snake-game/`
- **特色**: Canvas绘制、渐进式速度、键盘+触屏控制
- **技术难点**: 碰撞检测、食物生成算法、蛇身增长逻辑

### 2. ⚫⚪ 五子棋 (Gobang)
- **路径**: `gobang/`
- **特色**: 15x15标准棋盘、四方向胜负检测、悔棋功能
- **技术难点**: 胜负算法、棋盘坐标转换、状态管理

### 3. 🔢 2048游戏
- **路径**: `2048/`
- **特色**: 4x4网格滑动合并、平滑动画、移动端支持
- **技术难点**: 移动算法、合并逻辑、动画同步

### 4. 💣 扫雷 (Minesweeper)
- **路径**: `minesweeper/`
- **特色**: 三种难度、递归展开、经典Windows风格
- **技术难点**: 递归算法、首次点击保护、边界检测

### 5. 🧠 记忆翻牌 (Memory Game)
- **路径**: `memory-game/`
- **特色**: 3D翻转动画、16种emoji、时间统计
- **技术难点**: Fisher-Yates洗牌、CSS 3D变换、状态管理

### 6. 🐦 Flappy Bird
- **路径**: `flappy-bird/`
- **特色**: 重力物理、精确碰撞、随机管道生成
- **技术难点**: 物理引擎、AABB碰撞检测、游戏循环优化

### 7. 🧩 俄罗斯方块 (Tetris)
- **路径**: `tetris/`
- **特色**: 7种经典方块、等级递增、下一块预览
- **技术难点**: 矩阵旋转、行消除算法、碰撞系统

### 8. 🔢 数独 (Sudoku)
- **路径**: `sudoku/`
- **特色**: 三种难度、回溯算法、笔记模式、提示求解
- **技术难点**: 回溯生成算法、约束满足问题、数独求解器

## 🛠️ 技术架构

### 核心技术栈
- **HTML5 Canvas** - 游戏渲染引擎
- **CSS3** - 现代化UI样式和动画
- **原生JavaScript ES6** - 游戏逻辑和交互

### 设计原则
- **零依赖** - 不使用任何外部框架或库
- **响应式** - 完美适配桌面和移动设备
- **模块化** - 清晰的代码结构便于维护
- **性能优化** - 60fps流畅运行体验

### 通用功能
- ✅ 本地存储高分记录
- ✅ 暂停/继续游戏
- ✅ 键盘和触摸双重控制
- ✅ 游戏状态管理
- ✅ 响应式UI设计
- ✅ 跨浏览器兼容

## 🔧 开发说明

### 环境要求
- 现代浏览器(支持ES6+ 和 Canvas)
- 本地HTTP服务器(推荐Python、Node.js或Live Server)

### 调试模式
每个游戏都支持开发者工具调试：
1. 打开浏览器开发者工具(F12)
2. 在Control台查看游戏状态日志
3. 使用断点调试游戏逻辑

### 自定义修改
所有游戏都采用模块化设计，可以轻松修改：
- **游戏参数**: 修改JS文件中的配置对象
- **视觉样式**: 修改CSS文件中的颜色和布局
- **游戏逻辑**: 扩展JS类的方法
