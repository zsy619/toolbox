# 工具箱项目

一个综合性的在线工具集合,包含彩票工具、图像处理、实用工具、游戏集合等。

## 📁 项目结构

```
工具箱/
├── apps/                  # 应用工具集合
│   ├── lottery/          # 福彩工具
│   ├── ai-tools/         # AI工具
│   ├── image-processing/ # 图像处理
│   ├── games/            # 游戏集合
│   ├── utilities/        # 实用工具
│   ├── multimedia/       # 多媒体工具
│   ├── layouts/          # 布局工具
│   ├── demos/            # 演示项目
│   └── tests/            # 测试页面
├── assets/               # 静态资源
│   ├── css/             # 样式文件
│   ├── js/              # JavaScript文件
│   └── fonts/           # 字体文件
├── scripts/              # 脚本工具
├── docs/                 # 项目文档
│   ├── page-docs/       # 页面文档
│   ├── guides/          # 开发指南
│   └── image-guides/    # 图像处理指南
├── config/               # 配置文件
├── archive/              # 归档目录
├── index.html            # 主导航页面
└── README.md            # 本文件
```

## 🚀 快速开始

从 [index.html](index.html) 开始使用所有工具。
