## 🔧 配置说明

### 修改排除目录

编辑 `scripts/generate-apps-index.py`:

```python
EXCLUDE_DIRS = {
    '.git', '.vscode', 'node_modules', '__pycache__', 
    'games', 'archive', '.idea',  # 添加要排除的目录名
}
```

### 修改文件图标映射

```python
FILE_ICONS = {
    'html': 'fa-file-code',
    'pdf': 'fa-file-pdf',
    'zip': 'fa-file-archive',
    # 添加更多文件类型...
}
```

## ⚠️ 注意事项

1. **覆盖现有index.html**
   - 脚本会覆盖已存在的 index.html
   - 如果需要自定义首页,请勿运行此脚本

2. **资源路径依赖**
   - 确保 `assets/` 目录存在
   - 确保 LayUI 和 FontAwesome 文件可用

3. **games目录**
   - games目录包含100+个游戏文件
   - 已在配置中排除,未生成索引
   - 如需生成,请修改 `EXCLUDE_DIRS` 配置
