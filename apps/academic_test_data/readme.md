## 问题列表

### 1、 默认账户与密码

注意：省、市、学校导入以后的账号是登记的手机号，密码与手机号相同，可登录后修改。

### 2、条码下载要选择科目吗

需要选择：不同的科目的人是不一样

例如：语文可能有1000人考试，物理就不一定了啊

### 3、条码打印

网上下载的条形码，需要专用的打印纸，每张40个难分割，并不好买到

### 4、考生信息导入

（1）下载模板，按照模板要求填

（2）要求考生号为14位

### 5、同考场小语种限制

于2026年3月20日10点已放开同考场小语种限制

### 6、考生数据更新

（1）需要删掉原来的并重新导入

（2）否则系统提示重复

### 7、平台支持多端吗

目前只支持电脑端，方便资料的整理及上传。

### 8、提示考生号重复

首先：排查excel数据是否正确，与原始数据比对

其次：联系技术管理员，提供重复的考号在平台中核对
