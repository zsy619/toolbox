# 图像处理工具

## 工具列表

| 文件 | 功能 |
|------|------|
| idphoto.html | 证件照处理 |
| image_edit.html | 图像编辑 |
| image_merge.html | 图像合并(标准版) |
| image_splitter.html | 图像分割 |
| layui_image_merge.html | LayUI图像合并 |
| layui_image_merge_optimized.html | LayUI图像合并(优化版) |
| photo_separation.html | 照片分离 |

## 技术栈
- LayUI 2.6.8 / 2.8.3
- FontAwesome 6.0.0 / 6.4.0
- Cropper.js (证件照)
- JSZip + FileSaver (照片分离)
